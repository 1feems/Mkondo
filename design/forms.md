# FlowOS — Figma Forms Reference

What operators and builders see on screen. Use this to design each form in Figma — every label, placeholder, and helper text in order.

---

## Form 1 — Program Info

**Page Title:** Program Info

**Page Description:** This form sets the identity of your grant program. Filled once and applies to all rounds. Appears in the registry and on your public application page.

---

**Program Name**
Placeholder: Enter your program name
Helper: e.g. Stacks Builder Grant

**Ecosystem / Chain**
Placeholder: Enter ecosystem or chain
Helper: e.g. Stacks, Optimism, Near

**Program Description**
Placeholder: Describe your program
Helper: What is this program trying to achieve? Describe the gap or opportunity it addresses.

**Funding Type**
Static value: Grant Program

**Mechanism**
Dropdown — default: Grant
Options: Grant

**Program Focus**
Label: Select all that apply

User Outcomes
☐ Acquisition
☐ Retention
☐ Activation
☐ Reactivation

Ecosystem Outcomes
☐ Security
☐ Infrastructure
☐ Interoperability
☐ Developer Tools
☐ Ecosystem Growth

**Vertical**
Dropdown — Placeholder: Select vertical
Options: DeFi / Protocols, Consumer Apps / Wallets

**Eligibility Criteria**
Label: Define the conditions a project must meet to qualify

  Criterion 1
  Placeholder: e.g. Project must be live and deployed on supported chain

  Criterion 2
  Placeholder: e.g. Minimum 3 months of demonstrable user activity or traction

  Criterion 3
  Placeholder: e.g. No major security incidents in the last 6 months

  + Add criterion (optional, up to 2 more)

**Expected Deliverables**
Placeholder: Describe expected outputs
Helper: What outputs do you expect from funded projects? e.g. integrations, tooling, user campaigns

**Rubric**
Dropdown — default: User Outcome Rubric
Options: User Outcome Rubric

**Program Lead / Contact**
Placeholder: Full name or team name
Helper: Who is the internal owner or point of contact for this program?

**Primary Impact Goal** (optional)
Placeholder: e.g. Grow active users via high-usage integrations
